רווח/הפסד לפי מניה
האפליקציה מציגה ניתוח אוטומטי בהתבסס על הכללים (TP/SL) שהזנת — זהו כלי עזר בלבד ולא ייעוץ השקעות. גם פאנל "תחזית ומידע אנליסטים" בכל פוזיציה הוא מידע אינפורמטיבי שמסוכם מנתונים פומביים (דירוגי ויעדי מחיר של אנליסטים), ולא המלצת קנייה/מכירה אישית — הוא עלול להתעדכן באיחור ואינו תחליף לבדיקה עצמאית. ההחלטה הסופית על קנייה, מכירה או שינוי יעדים היא שלך בלבד, ואני לא יועץ השקעות מורשה. מחיר "נוכחי" מתעדכן אוטומטית בסיום יום המסחר ממקור נתוני שוק (כש מחובר), וניתן גם לעדכן אותו ידנית בין לבין — עדכון ידני יוחלף בסנכרון הרשמי הבא.

☁ סנכרון ענן בין מכשירים
הגדרה חד-פעמית לכל מכשיר. הנתונים ישמרו ב-Supabase (חינמי) ויסונכרנו אוטומטית.
שלב 1 — צור טבלה ב-Supabase
היכנס ל-supabase.com → הפרויקט שלך → SQL Editor → הרץ:
create table if not exists portfolio ( id text primary key default 'default', positions jsonb not null default '[]'::jsonb, updated_at timestamptz default now() ); alter table portfolio enable row level security; create policy "anon_all" on portfolio for all using (true) with check (true);
